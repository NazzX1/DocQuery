# DocQuery

This project leverages AI to efficiently search, retrieve, and answer questions from documents. It processes text chunks, embeds them for similarity search, and provides contextual responses based on the user's query.

## Technologies Used
- **Python**
- **LangChain**
- **Chroma**
- **Ollama**


## How to Run
1. Clone the repository.
2. Install the required dependencies:
    ```
    pip install -r requirements.txt
    ```
3. Set up environment variables (e.g., API keys).
4. Run the application:
    ```
    python main.py
    ```

## Planned Features
- **Multi-format Support**: Extend the document upload functionality to support additional formats like Word and Excel.
- **OCR Integration**: Allow users to upload images and extract text from them, enabling queries based on the text found in those images.
